# **App Name**: LeaseLoop

## Core Features:

- Secure User Authentication & Role Management: Enable secure login for both landlords and tenants, directing them to their respective role-specific dashboards (Landlord Dashboard or Resident Hub).
- Property & Document Management: Allow landlords to add, edit, and view property details, and to upload/manage property images and critical documents, securely stored in a Supabase bucket.
- Tenant Management: Landlords can add new tenants, manage their information, and assign them to specific properties within their portfolio.
- Maintenance Request & Tracking System: Tenants can submit and view the status of their maintenance requests, while landlords can log, track status (e.g., pending, in progress, completed), and manage all maintenance tasks.
- Inspection Scheduling & PDF Report Generation: Landlords can schedule property inspections, record findings directly within the platform, and generate downloadable PDF reports of the inspection outcomes.
- Secure Tenant Document Hub: Tenants can securely view details of their assigned property and access important documents, including generated emergency contact sheets.
- AI-Powered Maintenance Request Triage: A generative AI tool that assists landlords by analyzing new tenant maintenance requests, automatically suggesting priority levels and categorizing issues to streamline response times and workflow.

## Style Guidelines:

- Primary color: A sophisticated, muted blue (#2D4E85), symbolizing reliability and professionalism. It offers a strong contrast against lighter elements.
- Background color: A soft, almost ethereal blue-grey (#F0F3F8), providing a clean and spacious canvas that harmonizes with the primary color and supports readability.
- Accent color: A vibrant yet elegant violet (#8B72DA), chosen for its dynamic contrast, used to highlight key interactive elements and calls to action.
- Headlines: 'Space Grotesk', a modern sans-serif, chosen for its tech-forward and clear presentation in titles and important announcements.
- Body text: 'Inter', a highly readable grotesque-style sans-serif, ensuring optimal legibility for detailed property and maintenance information across all screen sizes.
- Utilize a consistent set of clean, modern line icons for navigation and functional controls. These will offer clarity and intuitive understanding without adding visual weight.
- Implement a clean, modular layout with clear separation between content blocks. Dashboards will leverage a responsive grid system to effectively display information hierarchy on both desktop and mobile devices.
- Incorporate subtle, tasteful micro-animations for interactive elements and state changes. This provides instant user feedback and enhances the overall polish of the interface without being distracting.
